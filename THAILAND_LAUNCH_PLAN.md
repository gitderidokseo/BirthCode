# 태국어 런칭 계획

## 1. 배경 / 시장 근거

- 태국은 한국식 "사주팔자" 개념 자체는 낯설지만, 중국식 "ปาจื้อ(Bazi)" 문화로 사주와 유사한 운세 소비 문화가 이미 존재함.
- 태국어로 서비스하는 **AI 기반 한국식 사주 앱**은 경쟁 조사 기준 확인되지 않음 — 니치 존재.
- 인앱결제 시장 지표:
  - 게이머 1인당 연 인앱결제 지출 **393달러**, 동남아 1위
  - 유료 결제 경험자 비율 **49%** (북미 5~10% 대비 매우 높음)
  - 모바일 뱅킹/핀테크 앱 사용률 조사 대상국 중 1위(65%) — 결제 인프라 성숙
- 결론: "그냥 번역"이 아니라 "한국 사주 + AI"라는 정체성을 살린 현지화가 핵심 차별점.

## 2. 기술 작업

### 2.1 앱 내 다국어 추가
- [x] `app/src/main/assets/translations.js`에 `th` 키 추가 (ko와 동일한 99개 키 전체 번역, 문법 검증 완료)
- [x] `app/src/main/assets/script.js`의 언어 감지/폴백 로직에서 `th` 정상 매칭 확인 (코드 수정 불필요 — `translations[userLang]` 존재 여부로 자동 판별)
  ```js
  const userLang = (navigator.language || 'ko').split('-')[0];
  currentLang = translations[userLang] ? userLang : 'ko';
  ```
- [x] Android `res/values-th/strings.xml`, iOS `th.lproj/Localizable.strings`에 결제 관련 Toast 문구(취소/오류/상품 없음 등) 태국어 추가
- [x] iOS `project.yml`에 `th.lproj` 리소스 경로 + `CFBundleLocalizations`에 `th` 등록 (XcodeGen 빌드 반영용, 안 하면 빌드에서 누락됨)
- [ ] **실제 기기/에뮬레이터에서 태국어 UI로 빌드해서 눈으로 확인** (아래 5번 항목 참고)
- [x] `interpretation.js` / `manseryeok.js` 점검 완료 — 한국어 텍스트는 전부 주석/JSDoc이거나 `hangul: '갑'` 같은 계산용 내부 데이터. 사주 기둥(갑/을/자/축 등)은 모든 언어에서 공통으로 한자/한글 그대로 표시하는 기존 설계라 태국어만의 추가 작업 불필요.

### 2.2 인앱결제 가격 설정 (Play Console)
- 4개 상품(`saju_haiku`, `saju_sonnet`, `saju_opus`, `saju_fable`)에 대해 태국 국가별 가격을 **자동 환산에 맡기지 않고 수동 오버라이드**
- 1차 제안가 (원가 대비 마진 검증 후 재조정):

| 등급 | 한국 가격 | 태국 제안가 | 비고 |
|---|---|---|---|
| Haiku | 500원 | 9~12 THB | 원가($0.06) 대비 마진 충분 → 진입장벽 낮추는 체험 티어로 활용 |
| Sonnet | 1,000원 | 25~29 THB | Sonnet 실제 API 원가 확인 후 재조정 필요 |
| Opus | 1,500원 | 39 THB | 동일 |
| Fable | 2,000원 | 59 THB | 동일 |

- [ ] Sonnet/Opus/Fable 실제 API 원가 확인 → 마진표 재계산
- [ ] Play Console에서 태국 가격 수동 입력

## 3. 스토어 등록정보 현지화 (Play Console)

- [ ] "스토어 등록정보 → 번역 관리"에서 태국어(ไทย) 리스팅 신규 추가
- [ ] 앱 이름(브랜드)은 유지, 태국어 리스팅의 **부제/짧은 설명에 ASO 키워드 반영**
  - 키워드 후보: `AI`, `เกาหลี(한국)`, `ซาจู`/`ปาจื้อ`, `ดูดวง`
  - 예시 패턴: `[브랜드명] – เกาหลีซาจู AI ทำนายดวง`
- [ ] 전체 설명(long description) 태국어 번역
- [ ] 스크린샷을 태국어 UI 기준으로 새로 캡처 (한국어 스크린샷 재사용 금지)
- [ ] 태국어 검색 키워드 리서치 (실제 태국 사용자가 쓰는 검색어 확인)

## 5. 참고 — 향후 확장 후보 (태국 다음)

- **일본**: 사주와 동일한 개념(四柱推命)이 이미 존재, 점술 시장 규모 약 1조 엔. 단, 한국 개발사의 AI 사주 앱(`占いGPT`, kr.semaphore.sajugpt)이 이미 선점 중이라 차별화 포인트 필요.
- **베트남**: Tử vi 문화 기반 유사 운세 소비 강함, 인앱결제 지출 동남아 2위(연 284달러). 경쟁 앱 중 "한국식" 포지셔닝 확인 안 됨 — 상대적으로 빈 시장.
